

## Description

Torob Clone Backend built with nestjs & postgresql <br>
<a href="https://github.com/MohammadHoseinDadashi/torob-frontend">Frontend Repo</a>

## Screenshot

![torob clone](https://github.com/MohammadHoseinDadashi/torob-backend/blob/main/1.png)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
